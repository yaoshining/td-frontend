$modal是一个可以快速创建AngularJS强大的模态窗口的服务.创建它们是非常直接的:创建一个区域视图,它们的控制器以及当使用这个服务时去引用它们.

$modal服务紧紧只有一个方法:open(options),这里可用的选项如下:

  • templateUrl - 一个可以代表模态窗口内容的模板路径.
  • template - 可以代表模态窗口内容的内置模板.
  • scope - 一个可以被模态窗口内容使用的作用域实例(实际上$modal服务将会创建一个被提供作用域的子作用域).默认值是$rootScope
  • controller - 一个模态窗口实例的控制器 - 它可以初始化模态窗口要使用的作用域.支持像这种'SomeCtrl as myctrl'的"controller-as"语法;可以同 $modalInstance一起注入.
  • controllerAs - 另一种"controller-as"语法,和指定定义的API使用方法一致.需要提供controller选项.
